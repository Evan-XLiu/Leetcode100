# 283. [移动零](https://leetcode.cn/problems/move-zeroes/?envType=study-plan-v2&envId=top-100-liked)

## 题目描述
* 给定一个数组 `nums`，编写一个函数将所有 `0` 移动到数组的末尾，同时保持非零元素的相对顺序。
* 请注意，必须在不复制数组的情况下原地对数组进行操作。

### 示例
* **示例 1:** 输入: nums = [0,1,0,3,12] 输出: [1,3,12,0,0]
* **示例 2:** 输入: nums = [0] 输出: [0]

### 限制
* 1 <= nums.length <= 10^4
* -2^31 <= nums[i] <= 2^31 - 1
* **进阶：** 你能尽量减少完成的操作次数吗？

## 解法思路
* 本题最优且最清晰的解法是**双指针法**。
* 我们维护两个指针：左指针（`left`）和右指针（`right`）。左指针的作用是标记当前已经处理好的非零序列的尾部（也就是下一个非零元素应该放置的位置），而右指针的作用是遍历整个待处理的数组。
* 随着右指针不断向右移动，只要右指针遇到了一个非零数，我们就把这个非零数和左指针指向的数进行交换，并且把左指针向右移动一位。
* 由于左指针左边一定都是非零数，而左指针到右指针之间的数一定都是被略过的零，因此每次交换仅仅是将位于前面的零和位于后面的非零数互换，完美地保证了非零数原有的相对顺序不受影响。

### 算法步骤
* **步骤 1:** 初始化左指针 `left` 和右指针 `right`，起始位置均设为 0（数组头部）。
* **步骤 2:** 使用循环让右指针 `right` 遍历整个数组。
* **步骤 3:** 在遍历过程中，判断 `nums[right]` 的值。如果不等于 0，则将 `nums[left]` 与 `nums[right]` 进行值交换。
* **步骤 4:** 发生交换后，左指针 `left` 向右步进 1（`left += 1`），准备接收下一个非零数。
* **步骤 5:** 无论是否执行了交换操作，右指针 `right` 都要向右步进 1（`right += 1`），直到检查完数组的所有元素。

### 复杂度分析
* **时间复杂度:** O(n)，其中 n 为数组 `nums` 的长度。数组中的每个元素最多被左右指针分别访问一次，属于线性时间复杂度。
* **空间复杂度:** O(1)。算法在执行过程中只使用了两个额外的指针变量，直接在原数组上进行原地修改，没有申请额外的内存空间。

## 代码实现

### python
```python
class Solution:
    def moveZeroes(self, nums: list[int]) -> None:
        """
        Do not return anything, modify nums in-place instead.
        """
        n = len(nums)
        # 初始化双指针
        left = 0
        right = 0
        
        # 右指针负责遍历整个数组
        while right < n:
            # 遇到非零元素时，执行交换操作
            if nums[right] != 0:
                # 将非零元素按顺序换到左边，并将零换到右边
                nums[left], nums[right] = nums[right], nums[left]
                # 左指针右移，指向下一个待放置非零元素的位置
                left += 1
            
            # 右指针继续向右探索下一个元素
            right += 1
```
